# Neon Purple Portfolio Website

This is a responsive multi-page personal portfolio website built with HTML, CSS, and JavaScript. The website features a neon purple aesthetic and is designed to work well on both desktop and mobile devices.

## Website Structure

The portfolio consists of three main pages:

1. **index.html** - Home page with:
   - Personal introduction and profile photo
   - Education timeline
   - Work experience
   - Skills section with visual skill bars
   - Contact form and information

2. **projects.html** - Projects showcase page with:
   - Filterable project gallery
   - Project cards with images and descriptions
   - Technology tags for each project

3. **blog.html** - Blog page with:
   - Article cards with images and excerpts
   - Publication dates and reading times
   - Pagination navigation
   - Newsletter subscription form

## Design Features

- **Neon Purple Theme**: Consistent color scheme with neon purple accents
- **Responsive Layout**: Adapts to all screen sizes using CSS media queries
- **Interactive Elements**: Hover effects, animations, and interactive filtering
- **Clean Navigation**: Fixed header with mobile hamburger menu
